audio_prbs_genchk_drv.c File Reference

Overview

This file contains the driver implementation for the Audio PRBS generator and checker used in the HDMI example design.

Parameters

Returns
  • TRUE if checker is enabled
  • FALSE if checker is disabled
Note
The checker is enabled after the enable bit is set and after a start of audio block has been seen.
